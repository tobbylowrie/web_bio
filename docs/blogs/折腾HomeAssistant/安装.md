# 安装 Home Assistant

Home Assistant 有很多种版本

以在PVE中安装 OS 版为例


## 0.环境

镜像包：[https://fw.wghaos.com/haos/x86-64-pc/win10pex86-wg-haos15.2.iso](https://fw.wghaos.com/haos/x86-64-pc/win10pex86-wg-haos15.2.iso)



## 1. 下载镜像文件

由于国内特殊网络环境，不建议直接下载并使用官方镜像进行安装，安装过程将会十分折磨

建议使用国内冬瓜大佬的镜像：[官网链接](https://www.wghaos.com/)
